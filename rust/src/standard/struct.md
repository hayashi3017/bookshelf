# struct

## impl

メソッドは構造体定義の中ではなく、implブロックに記述する。

- 関連関数
  - implブロック内に定義した関数
  - 呼び出される対象の値`Self`を最初の引数としなければならない
- 自由関数
  - implブロック外に定義した関数
- 型関連関数
  - 特定の型に対するimplブロック内に定義した関数
  - `self`を引数として持たない
  - 例
    - コンストラクタ
      - ちなみにコンストラクタ関数をnewとするのは慣習
  - メソッドと区別される
  - `Foo::new`という形で使用する

## 定数

- 型関連定数
  - インスタンスを参照せずに使用できる
  - `Foo::T`という形で使用する
